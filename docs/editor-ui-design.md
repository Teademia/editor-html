# 编辑器 UI 设计文档

## 总体布局

```
┌─────────────────────────────────────────────────────────────────────┐
│  TOOLBAR  [全文] [结构] [节点图]          [保存] [导出.dtl] [▶预览]  │
├──────────────┬──────────────────────────────────┬───────────────────┤
│              │                                  │                   │
│   L2         │   CENTER PANE                    │   AI 面板         │
│   结构时间线  │   （随 Tab 切换内容）             │   （上下文感知）   │
│              │                                  │                   │
│   [块1]      │                                  │                   │
│   [块2]      │                                  │                   │
│   [块3] ←选中│                                  │                   │
│   [块4]      │                                  │                   │
│   ...        │                                  │                   │
│              │                                  │                   │
│   [+ 新建块] │                                  │                   │
│              │                                  │                   │
└──────────────┴──────────────────────────────────┴───────────────────┘
```

**三栏固定结构：**
- **左栏**：L2 结构时间线，始终可见，是导航主干
- **中栏**：随顶部 Tab 切换（全文 / 块详情 / 节点图）
- **右栏**：AI 面板，根据当前所在层级自动切换内容

## 左栏：L2 结构时间线

```
┌──────────────┐
│ 📖 故事结构   │
│ 20万字 · 10块 │
├──────────────┤
│ ━━ 第一幕 ━━ │
│ ┌──────────┐ │
│ │ 块 1     │ │
│ │ 引入世界  │ │
│ │ 2万字    │ │
│ │ ● 2选择点│ │
│ └──────────┘ │
│ ┌──────────┐ │
│ │ 块 2     │ │
│ │ 初次失败  │ │
│ │ 2万字    │ │
│ └──────────┘ │
│ ━━ 第二幕 ━━ │
│ ┌──────────┐ │  ← 选中状态（高亮）
│ │▶块 3     │ │
│ │ 盟友背叛  │ │
│ │ 2万字    │ │
│ │ ⚠ 待生成  │ │
│ └──────────┘ │
│   ...        │
│              │
│ ◆ 分叉点     │  ← 特殊标记
│   块9 开始   │
│              │
│ [+ 新建块]   │
└──────────────┘
```

**块卡片状态标记：**
- `✓ 已完成`：骨架已生成，.dtl 已导出
- `⚠ 待生成`：块设计填写完毕，骨架未生成
- `✏ 草稿`：块设计未完成
- `◆ 分叉点`：作者标记的结局路由节点

**交互：**
- 点击块 → 中栏切换为该块的 L3 详情
- 拖拽块 → 调整顺序
- 右键块 → 复制 / 删除 / 标记为分叉点

## 中栏 Tab 3：节点图（L4 Scene Graph）

现有的 Drawflow 编辑器，但增加了块上下文显示。

```
┌────────────────────────────────────┐
│ 节点图 · 块3 盟友的背叛             │
│ [← 所有块] [显示全部块] [仅本块]    │
├────────────────────────────────────┤
│                                    │
│  ╔══════╗    ╔══════╗              │
│  ║ 录像室 ║──►║ 走廊  ║            │
│  ╚══════╝    ╚══╤═══╝              │
│                 │                  │
│            ╔════▼═══╗              │
│            ║ 质问场景 ║             │
│            ╚════╤═══╝              │
│          ┌──────┴──────┐           │
│      ╔═══▼══╗      ╔═══▼══╗       │
│      ║ 对峙  ║      ║ 隐忍  ║      │
│      ╚══════╝      ╚══════╝       │
│                                    │
│  [+ AI生成场景]  [导入骨架要点]      │
└────────────────────────────────────┘
```

**新增功能（相比现有编辑器）：**
- 顶部显示当前所属块
- `[导入骨架要点]`：将 L3 骨架的每个分点自动创建为 label 节点
- `[AI 生成场景]`：选中节点，右侧 AI 面板生成该场景的台词/旁白

## 导航流程

```
启动编辑器
    │
    ▼
L1 全文（填写 Story Bible）
    │ AI 检查通过
    ▼
L2 结构（生成/规划块列表）
    │ 点击某块
    ▼
L3 块详情（填写块设计 → AI 生成骨架 → 审批选择点）
    │ 点击「生成节点图」
    ▼
L4 节点图（骨架要点 → label 节点 → AI 生成台词）
    │ 完成本块
    ▼
返回 L2，处理下一块
    │ 所有块完成
    ▼
导出 .dtl → Godot / Dialogic 运行
```

## 数据存储结构

```json
{
  "story_bible": { ... },
  "blocks": [
    {
      "id": "block_3",
      "title": "盟友的背叛",
      "act": "second_first_half",
      "status": "skeleton_done",
      "design": { ... },
      "skeleton": "1. 深夜苏晨...\n2. 走廊撞见...",
      "input_state": { ... },
      "output_state": { ... },
      "scenes": ["block3_replay", "block3_corridor", ...]
    }
  ],
  "scenes": {
    "block3_corridor": "label block3_corridor\n走廊的灯...\n"
  },
  "branching_point": {
    "position": "block_9",
    "routes": [ ... ]
  }
}
```

整个项目存储为单个 `.vnproject` JSON 文件，
导出时将 `scenes` 拼合为 `story.dtl`。

## 实现顺序

```
Sprint 1：框架搭建
  ✓ 三栏布局（左/中/右）
  ✓ 左栏：块列表 + 分幕分组
  ✓ 中栏：Tab 切换
  ✓ 右栏：AI 面板空壳

Sprint 2：L1 + L2
  ○ Story Bible 表单
  ○ 块卡片设计（状态、拖拽排序）
  ○ 分叉点标记

Sprint 3：L3 骨架生成
  ○ 块详情面板
  ○ AI 骨架生成接入 API
  ○ 选择点审批流程

Sprint 4：L4 集成
  ○ 骨架要点 → label 节点自动创建
  ○ 场景级 AI 台词生成
  ○ 与现有 Drawflow 编辑器对接

Sprint 5：导出
  ○ .vnproject 文件保存/加载
  ○ 导出为 .dtl
  ○ 一致性校验（伏笔账本、变量覆盖）
```
