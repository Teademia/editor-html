# 视觉小说要素清单

## 核心要素（MVP 必需）

### 1. 叙事结构层

| # | 要素 | 说明 | .dtl 支持 |
|---|---|---|---|
| 1 | 场景 | 故事发生的地点与时间节点 | `label` |
| 2 | 场景人物 | 哪些角色在场 | `join` / `leave` |
| 3 | 场景对话 | 角色台词与旁白 | `角色: 台词` / 旁白行 |
| 4 | 场景顺序 | 场景之间的推进关系 | `jump` |
| 5 | 玩家选择 | 分支选项文字与条件 | `- 选项 \| [if 条件]` |
| 6 | 选择影响 | 选择对变量/旗标的改变 | `set {变量} op 值` |
| 7 | 结局条件 | 变量组合触发不同结局 | 多个 `end` label + 条件跳转 |
| 8 | 旗标/标志位 | 隐性剧情开关（见过某人、知道某秘密） | `set {flag} = true` |

### 2. 角色层

| # | 要素 | 说明 | .dtl 支持 |
|---|---|---|---|
| 9 | 角色好感度/关系值 | 数值化的角色关系，影响分支 | `set {affection} += 1` |
| 10 | 角色立绘状态 | 不同场景对应的表情/服装 | `join 角色 (表情)` |
| 11 | 角色出入场时机 | 谁在场、谁不在场、站哪个位置 | `join` / `leave` / transform |

### 3. 视听层

| # | 要素 | 说明 | .dtl 支持 |
|---|---|---|---|
| 12 | 背景图/CG | 每个场景对应的背景或事件图 | `background` 事件（需扩展） |
| 13 | BGM | 场景情绪对应的背景音乐 | `music` 事件（需扩展） |
| 14 | 音效 | 特定台词/事件触发的音效 | `sound` 事件（需扩展） |
| 15 | 文字演出 | 震动、渐显、打字机速度等特效 | Dialogic 样式标签 |

### 4. 系统层

| # | 要素 | 说明 | 优先级 |
|---|---|---|---|
| 16 | 存档点设计 | 玩家可以在哪里存档 | 中 |
| 17 | 回想/图书馆 | 已解锁 CG、BGM、对话回放 | 低 |
| 18 | 多周目设计 | 二周目解锁内容、速读跳过 | 低 |
| 19 | 时间线/章节结构 | 场景的宏观分组顺序 | 中 |

---

## MVP 范围建议

```
要素 1-8   叙事结构   ← 必须实现
要素 9-11  角色层     ← 尽量实现
要素 12-13 背景+BGM   ← 尽量实现
要素 14-19 其余       ← 毕设后续迭代
```

---

## AI 辅助切入点

| 要素 | AI 能做什么 |
|---|---|
| 场景 + 对话（1-3） | 给定世界观/人设，自动生成台词和旁白 |
| 场景顺序（4） | 给定故事梗概，生成场景骨架（label 结构） |
| 玩家选择（5） | 给定场景上下文，生成有意义的选项 |
| 选择影响（6） | 推荐合理的变量设计和数值变化 |
| 结局条件（7） | 根据变量体系推荐结局分支设计 |
| 角色立绘（10） | 根据台词情绪推荐表情标签 |
| BGM（13） | 根据场景氛围推荐音乐风格描述 |

---

## 输出格式

所有要素最终输出为 Dialogic 2 兼容的 `.dtl` 文件，
可直接导入 Godot 游戏本体运行，无需转换。
